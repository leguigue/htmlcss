Les bases du développement.

Introduction au coding.

Si vous vous êtes retrouvé sur ce site c'est sûrement car le développement web vous intrigue. Tout d'abord vous êtes en droit de vous questioner,sur ce qu'est le développemnt web. Le développement web consiste à créer des sites et des applications accessibles via Internet. Pour construire ces sites, on utilise principalement trois technologies de base : HTML, CSS et JavaScript. Chacune de ces technologies joue un rôle distinct et crucial dans la création d'un site web fonctionnel et esthétique. Nous allons vous introduire a ce milieu en vous presentant les outils basique utilisé par les développeur dans le cadre de création de site web. Tout d'abord vous trouverais ci-bas une courte explication de ceux-ci. Pour retrouver une explication plus détaillé des outils veuillez vous reférer au onglets adéquats.

le HTML

Le HTML (HyperText Markup Language), est le langage de balisage standard utilisé pour créer la structure de base des pages web. Il permet de définir les éléments et le contenu de la page, tels que les titres, les paragraphes, les images, les liens, les listes, et bien plus encore. Chaque élément HTML est représenté par des balises, qui encadrent le contenu et lui donnent une signification. Par exemple, une balise < h1> est utilisée pour les titres principaux, tandis qu'une balise < p> représente un paragraphe.

Le CSS

Le CSS (Cascading Style Sheets), est le langage de feuille de style utilisé pour contrôler l'apparence et le design des pages web créées avec HTML. Il permet de spécifier les couleurs, les polices, les espacements, les mises en page et d'autres aspects visuels de la page. En séparant le style du contenu, CSS facilite la maintenance et la mise à jour des styles de plusieurs pages web de manière cohérente. Par exemple, vous pouvez utiliser CSS pour changer la couleur du texte d'un paragraphe en ajoutant une règle de style comme color: blue;.

Le JavaScript

Le JavaScript est le langage de programmation utilisé pour ajouter des fonctionnalités interactives aux pages web. Il permet de rendre les pages dynamiques et réactives en répondant aux actions des utilisateurs, telles que les clics, les saisies de formulaires, et les mouvements de la souris. JavaScript peut également être utilisé pour manipuler le contenu HTML et CSS, permettant des mises à jour en temps réel sans recharger la page. Par exemple, vous pouvez utiliser JavaScript pour afficher une alerte lorsque l'utilisateur clique sur un bouton.
